# Contributing

Thank you for your interest in improving Turabian-9th!

## Reporting Issues
- Report citation formatting problems with examples
- Include the source type (book, article, etc.)

## Submitting Improvements
1. Fork the repository
2. If needed use the [Official CSL Editor](https://editor.citationstyles.org/)
3. Test changes with Zotero/Juris-M
4. Submit PR with detailed description of changes
